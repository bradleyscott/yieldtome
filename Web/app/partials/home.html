Speakers Lists

Create Speakers Lists. Have delegates add themselves to speak on them from their phone, tablet or computer. Shuffle the order and never lose track

Voting

Create a Poll and ask delegates to cast their Vote. See the Votes come in in real time and see whether or not the Poll has passed or failed

Delegate lists

Create your own delegate Profile and share contact details, or, stay behind your delegate persona. Like other delegates- and find out if they like you too

We have lots of ideas and you can influence the priority by voting on our feature request site. Some of the ideas on the list are:
  • Delegate to delegate instant messaging
  • Add delegates to an Event in bulk. A time saver for event organisers
  • Keyboard shortcuts - for power users with a keyboard
  • iPhone and Android Apps
  • Help overlays to assist first time users
 yieldto.me is open source- meaning you can see the code and contribute to fixing bugs and developing enhancements or new features. The code is available on GitHub for you to look at, fork, and submit pull requests to. We also have an RESTful API that can be used to develop your own stand-alone Apps and sites using yieldto.me's data. Get in contact if you want an API key.
